zkID

Privacy-Preserving Identity Verification

Full Pipeline: Prepare + Show with Shared Blinds

1

Generate Test Case

Create an SD-JWT with selective disclosure claims, issuer key pair, and device key pair.

2

Precompute

Parse credential, build JWT circuit inputs, generate witness, and prove Prepare circuit via WASM.

3

Present Proof

Sign verifier nonce with device key, build Show circuit inputs, prove Show circuit, and reblind both proofs with shared randomness.

4

Verify Proofs

Verify both Prepare and Show proofs, validate shared commitment (comm_W_shared), and extract public values.

Powered by Spartan2 + Hyrax ZK-SNARK prover running in WebAssembly.

Circuits: jwt.circom (Prepare) + show.circom (Show) on secq256r1 field.
